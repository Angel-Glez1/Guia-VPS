# Administracion de procesos en un VPS


El comando que se ocupa para manipular los procesos del servidor es

```bash
systemctl
```

Con este comando puedes **reiniciar**, **recargar**, **ver status**, **parar**,**iniciar**, procesos que este corriendo nuestro el vps
Ejemplo: Base de datos, Nginx, Python etc
Regularmente se ocupa cuando haces cambio de configuraciones en cualquiera de los
paquetes que se instalen


## Reiniciar el VPS
```bash
reboot
```
